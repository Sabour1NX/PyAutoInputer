## 核心功能
✅ **自定义输入内容**：通过 `config.txt` 设置需要自动输入的文本和间隔时间；
✅ **10秒倒计时**：启动后预留时间切换到目标窗口，避免输入错位；
✅ **自动配置生成**：无配置文件时自动创建默认 `config.txt`，无需手动新建；
✅ **错误日志记录**：运行异常时自动生成 `log.txt`，方便排查问题；
✅ **简洁GUI操作**：仅需点击按钮即可编辑配置、启动输入，无需命令行；
✅ **输入逻辑优化**：采用复制粘贴+回车模式，适配大多数文本输入场景。

## 环境要求
- 系统：Windows 7/10/11（32/64位）；
- Python 3.7+（需安装完整版本，确保包含 tkinter 组件）；
- 权限：必须拥有管理员权限（运行程序时需选择“以管理员身份运行”）。
